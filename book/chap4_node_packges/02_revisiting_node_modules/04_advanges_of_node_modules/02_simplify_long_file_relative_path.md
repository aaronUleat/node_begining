## Simplify Long File Relative Paths

Supongamos que tiene un módulo foo / foo.js que proporciona una serie de utilidades que necesita utilizar en una variedad de diferentes
Lugares en su aplicación. En una sección bar / bar.js, deberías requerir una llamada requerida ('../ foo / foo.js'), y en
En una sección bas / nick / scott.js, tendrías una llamada requerida que requiere ('../../../ foo / foo.js'). En este punto, usted
debe preguntarse: “¿Es este módulo de foo autónomo?” Si es así, es un gran candidato para pasar a node_modules / foo /
index.js en la raíz de la carpeta de su proyecto. De esta manera, puede simplificar sus llamadas para que solo requieran ('foo') en todo
tu codigo.
